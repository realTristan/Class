1. Factor each of the following fully

A) f(x) = x^3 - 3x^2 - 10x + 24
  Correct Answer:  (x - 2)(x - 4)(x + 3)
B) f(x) = 4x^3 + 12x^2 - x - 15
  Correct Answer:  (x - 1)(2x + 3)(2x + 5)
C) f(x) = x^4 + 8x^3 + 4x^2 - 48x
  Correct Answer:  x(x - 2)(x + 4)(x + 6)
D) f(x) = 4x^4 + 7x^3 - 80x^2 - 21x + 270
  Correct Answer:  (x + 2)(x + 5)(4x - 9)(x - 3)
E) f(x) = x^5 - 5x^4 - 7x^3 + 29x^2 + 30x
  Correct Answer:  x(x + 2)(x + 1)(x - 3)(x - 5)
F) f(x) = x^4 + 2x^3 - 23x^2 - 24^x + 144
  Correct Answer:  (x - 3)(x - 3)(x + 4)(x + 4)
